# Подключение к веб-интерфейсу графа (FalkorDB Browser)

grafit поднимает встроенный веб-браузер графа FalkorDB. В нём можно визуально смотреть
узлы и связи и выполнять Cypher-запросы по любому залитому проекту.

## Открыть UI

После `grafit up` интерфейс доступен на:

```
http://127.0.0.1:6400
```

(порт `6400` на хосте проброшен в порт `3000` контейнера `grafit-falkordb`).

## Что вводить на экране входа

На странице **FalkorDB Browser → Manual Configuration**:

| Поле | Значение |
|------|----------|
| Host | `localhost` |
| **Port** | **`6379`** |
| Username | пусто (Default) |
| Password | пусто (Default) |
| TLS Secured Connection | выключено |

Нажми **Log in**, затем выбери граф проекта (напр. `bpm`).

> Дефолтные значения на форме (`localhost` / `6379` / Default / Default) уже правильные —
> обычно достаточно просто нажать **Log in**.

## Почему порт `6379`, а не `6399` (важно)

Это сбивает с толку, но порт именно **`6379`**:

- Веб-UI отдаётся контейнером на порту `3000` → проброшен на хост как **`6400`** (по нему
  ты и открываешь браузер).
- К самой БД UI подключается **изнутри контейнера**, где FalkorDB слушает родной
  **`6379`**. Проброс `6399 → 6379` нужен только внешним клиентам с хоста (CLI `grafit`),
  браузеру он не нужен.

Итого: для UI — Host `localhost`, Port `6379`. Для внешних инструментов с хоста —
`127.0.0.1:6399`.

## Аутентификация

Пароль на FalkorDB не настроен (`requirepass` пуст), порты привязаны к `127.0.0.1`
(только локально). Поэтому Username/Password оставляй пустыми (Default).

## Полезные запросы в UI

Выбрав граф проекта, можно выполнять Cypher. Примеры:

```cypher
// сколько узлов и рёбер
MATCH (n:Entity) RETURN count(n);
MATCH ()-[r:LINK]->() RETURN count(r);

// найти узел по имени
MATCH (n:Entity) WHERE n.label CONTAINS 'Publish' RETURN n LIMIT 25;

// узел и его соседи (визуализация)
MATCH (n:Entity {label:'ProcessVersionStateMachine'})-[r:LINK]-(m:Entity)
RETURN n, r, m;

// какие сообщества (кластеры) есть
MATCH (n:Entity) RETURN DISTINCT n.community_label LIMIT 50;
```

> Семантический (векторный) поиск удобнее из CLI/MCP: `grafit query "..."` или инструмент
> `grafit_search`. Веб-UI — для визуального обхода графа и точечных Cypher-запросов.

## Графы проектов

Список залитых графов виден в `grafit list` (или `GRAPH.LIST` в UI). Один именованный
граф на проект (имя = basename git-репо).
